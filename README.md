A Simple Ray Tracer:
===================

参考`《Ray Tracing from the Ground Up》`<br>
与`《Fundamentals of Computer Graphics》`<br>
实现了一个简单的Ray Tracer。<br>
实现了：<br>
* Phong模型 Lambertian模型 Whitted模型
* 抖动采样
* 针孔相机模型
* 投射阴影
* 支持的对象球体，平面，长方形，三角形和三角形网格
* Phong明暗处理
* 缩放变换和平移变换
* 层次包围盒(BVHs)
